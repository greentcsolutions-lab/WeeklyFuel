# WeeklyFuel 🔥

> **Stop stressing over daily numbers — your week, your rules.**

A beautiful, mobile-first weekly calorie & budget tracker. Log once a day (or once a week), see your rolling weekly progress, and stop obsessing over what you ate on Tuesday.

No accounts. No app store. No data sent anywhere. Everything lives in your browser.

---

## ✨ Features

- **Weekly budget** — set a single weekly target for calories or spending
- **Flexible logging** — log any day, any time, update whenever you want
- **Rolling stats** — daily average, days remaining, budget per remaining day
- **Week navigation** — browse past weeks to review history
- **Dual mode** — toggle between 🔥 Calorie tracking and 💵 Budget tracking
- **Dark / light theme** — dark by default, one-click toggle
- **Fully offline** — pure localStorage, no backend, no auth
- **Mobile-first** — designed for phone use first

---

## 🚀 Deploy in 60 Seconds

1. Create a new GitHub repository (e.g. `weeklyfuel`)
2. Upload the files maintaining this structure:
   ```
   weeklyfuel/
   ├── index.html
   ├── css/
   │   └── style.css
   ├── js/
   │   └── app.js
   └── README.md
   ```
3. Go to **Settings → Pages → Deploy from branch → `main` / `root` → Save**
4. Your app is live at `https://yourusername.github.io/weeklyfuel`
